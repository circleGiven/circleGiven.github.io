Typescript's Types

2018-01-17 16:59:10
Dev>Typescript

타입스크립트에는 아래와 같은 타입들이 있다.
기본적으로 타입스크립트에서 선언한 타입과 다른 타입의 값을 넣으면 컴파일 과정에서 에러가 발생한다.

boolean

true나 false값을 갖는다.



number

기본적으로 숫자값이외에 2진수나 8진수 등 ECMAScript 2015에 따라 지원한다.



string

문자값을 가지며 ', ", ` 모두 사용 가능하다.

특히 backtick/backquote(`)는 위 두개와 달리 이런식으로도 응용이 가능하다.

뿐만아니라 ${} 안에서 연산도 가능하다.



Array

배열은 두가지 방법으로 사용이 가능하다.

타입을 선언하고 뒤에 []를 붙이는 방법과

generic으로 선언하는 방법이 있다.



Tuple

자바스크립트에는 없는 타입이다.
기존 자바스크립트에서 배열내에 자유로운 타입이 올 수 있었다.

예를들어

이런식으로 number형과 string형 처럼 다른타입이 같은 배열에 있어도 에러가 발생하지 않았다.

그러나 Tuple을 사용하면 에러가 발생한다.

이처럼 타입이라기 보다 제한한다고 생각하면 될 것 같다.
이것을 사용하면, 런타임 컨파일 전에 체크가 가능하기 때문에 오류를 쉽게 발견 할 수 있다.


Enum

흔히 다른 언어에 있는 Enum타입이다.
데이터 유형을 만들때 편리하다.
