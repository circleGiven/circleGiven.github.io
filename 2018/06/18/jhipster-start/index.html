JHipster

JHipster란?


JHipster는 스프링부트와 Angular/React 웹 어플리케이션과 Spring 마이크로 서비스를 생성, 개발 및 배포하는 개발 플랫폼이다.

즉, 웹 어플리케이션과 또는 마이크로 서비스 아키텍처를 생성하여 통합하는 것이다.

  • Spring 부트로 서버 측에서 고성능의 강력한 Java 스택
  • 매끄럽고 현대적인 모바일 최초의 프론트 엔드로 각도, 반응 및 부트 스트랩
  • JHipster 레지스트리, Netflix OSS, ELK 스택 및 Docker가있는 강력한 마이크로 서비스 아키텍처
  • Yeoman, Webpack 및 Maven / Gradle을 사용하여 애플리케이션을 빌드하는 강력한 워크 플로우

https://www.jhipster.tech/
